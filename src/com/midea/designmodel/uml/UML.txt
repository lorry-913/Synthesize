虚线箭头------>表示依赖（使用） Dependency
实线________表示关联 Association
实线三角________三角  表示继承
虚线三角--------三角  表示实现
实线虚正方形    表示聚合
实线实正方形     表示组合

类与类之间关系：
        依赖：A类种用到B类（属性，参数，返回类型都可以）
        继承、
        实现、
        关联：类与类之间关系，依赖的特例（只有属性依赖），一对一和多对一
        聚合：整体和部分可以分开，例如
                        class 电脑{ 鼠标；键盘  }
        组合：整体和部分不分开
                        class 人{ 身份=new 身份；头=new 头  } 人一没属性都没
